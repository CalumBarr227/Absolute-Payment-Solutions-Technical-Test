# Task Management Web Application

This is a simple **Task Management Web Application** built using **ASP.NET Core**, **Razor Pages**, and **Entity Framework Core** with the **Code-First approach**

The application allows users to manage tasks by performing basic CRUD operations

## Features

- Create new tasks  
- View a list of tasks  
- Update existing tasks  
- Delete tasks  

## Technologies Used

- ASP.NET Core
- Razor Pages
- Entity Framework Core
- SQLite
- C#

## Description

The project demonstrates how to build a basic web application using ASP.NET Core and Razor Pages. Entity Framework Core is used with a code-first approach to create and manage the database based on the application models

## How to Run

1. Clone the repository
2. Open the project in Visual Studio or VS Code
3. Run the application
4. The database will be created using Entity Framework Core migrations if configured
